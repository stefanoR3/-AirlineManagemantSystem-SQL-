# Airline Management System - Database Design

## Overview
This project focuses on the design and implementation of a relational database for an airline management system. It was developed as part of the **Databases** course during my 2nd year of Computer Science. The goal was to create a structured environment to manage flights, passengers, and bookings efficiently.

## Tech Stack
* **Language:** SQL
* **DBMS:** PostgreSQL
* **Concepts:** Normalization, Data Integrity, Security, Relational Modeling.

## Learning Outcomes
Through this project, I improved my ability to:
* Translate real-world requirements into a logical database schema.
* Write clean and efficient SQL queries.
* Understand the importance of referential integrity in complex systems.
