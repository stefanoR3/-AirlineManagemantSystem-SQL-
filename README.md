# Airline Management system - Database Desing

## Overview
This project focuses on the desing and implementation of a relational database for an airline management system.
It was developed as part of the **Databases** course during my 2nd year of Computer Science.
The goal was to create a structured environment to manage flights, passengers and bookings efficiently.

## Tech Stack
* **Language:** SQL
* **DBMS:** [PostgreSql]
* **COncepts:** Normalization, Data Integrity, Security, Relational Modeling.

## Learning Outcomes
Trough this project, i improved my ability to:
Translate real-world requirements into a logical database schema.
Write clean and efficient SQL queries.
Understanding the importance of referential integrity in complex systems.
